Ts常见问题总结

Written by _weimou with ♥ on January 21, 2020 in Typescript

  • 1、从一组数组对象中拿到相应值，组成枚举类型
  • 2、找不到第三方模块时，有两种方法可以消除ts的报警
  • 3、根据key值，约束对应value的值类型

1、从一组数组对象中拿到相应值，组成枚举类型

const littleNumbers = [
    { a: 1, b: '123' },
    { a: 2, b: '22223' },
    { a: 3, b: '3123' },
] as const;  // remember 'as const'

// littleNumbers[number] 相当于下标
type aLittle = typeof littleNumbers[number]['b']
// aLittle = '123' | '22223' | '3123'
let useLittles: aLittle = '3123'

2、找不到第三方模块时，有两种方法可以消除ts的报警

// 1、在加载第三方模块内部声明
declare const yourModuleName: any;

// 2、在根目录新建 modules.d.ts
declare module 'yourModuleName';

3、根据key值，约束对应value的值类型

let source = {
    name: 'lisa',
    age: 19
}

type typeSource = typeof source

const getKey = <T extends keyof typeSource>(
    key: T, value: typeSource[T]
): typeSource[T] => {
    return value
}

// error 类型“"123"”的参数不能赋给类型“number”的参数。ts(2345)
let bingoAge = getKey('age', '123')   

// is ok
let bingoName = getKey('name', '123')
