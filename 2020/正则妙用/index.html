常用正则技巧

  • 只允许输入 can, man, fan 反之返回 false
    • /[cmf]an$/

  • 严格匹配以’an’结尾，且不以’d’ ‘r’或’p’开头的任何三个字母词，要求只能是三个字母词

    • /^[^drp]an$/
    • 如果想排除数字的话可以这样写 ^[^a-z]an$

  • 匹配在这三个字母词的范围区间内的字符串 ‘Ana’, ‘Bob’, ‘Cpc’

    • 在表达式中使用不同的范围(精确匹配而不是一味的使用a-zA-Z)
    • /[A-C][n-p][a-c]$/

  • 匹配字符中重复的字母，最少2个，最多5个 ‘wazzzzze’, ‘wazze’, ‘waze’

    • /waz{2,5}e$/

  • 可选字符 严格匹配包括’几’个 ‘文件(或复数)’ ‘?(符号)’, 注意空格

    • example: ‘1 file found?’ ‘2 files found?’ ‘21 files found?’ ‘not file found’
    • /^\d+ files? found\?$/

  • 空白后跟固定字符

    • example: ‘1. qwe’ ‘2. qwe’ ‘3. qwe’
    • /^\d\.\s+qwe/


  • 捕获组 匹配前缀为 file至.pdf的内容

    • example: ‘file_transform.pdf’ ‘file_21300.pdf’ ‘testfile.tmp.pdf’
    • /^(file.+)\.pdf$/
